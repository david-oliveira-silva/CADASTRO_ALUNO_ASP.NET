Execução do Projeto
------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Este projeto é uma aplicação ASP.NET Core MVC que requer a instalação e configuração do servidor Firebird para ser executada. Siga os passos abaixo.

1. Requisitos e Ferramentas
Você precisará das seguintes ferramentas instaladas:

IDE: Microsoft Visual Studio Community 2022.

SDK: .NET 6.0 ou superior.

SGBD: Servidor Firebird (v5.0)

DBeaver Community (Necessário para  executar os scripts SQL).

------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

2. Configuração do Banco de Dados
   
O projeto utiliza acesso puro (sem ORM) ao Firebird. É essencial que o esquema (tabelas) esteja configurado corretamente e que a connection string aponte para o local correto.

2.1. Criação do Esquema (Script SQL)
Você deve executar o script SQL abaixo no seu banco de dados Firebird, utilizando o DBeaver ou outra ferramenta de administração, antes de rodar a aplicação.

-- SCRIPT DE CRIAÇÃO DO BANCO DE DADOS (FIREBIRD)

-- TABELA CIDADES --
CREATE TABLE cidades (
    cidadeID integer GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    cidadeNome varchar(100) NOT NULL,
    cidadeUF varchar(2) NOT NULL
);

-- TABELA ALUNOS --
CREATE TABLE alunos (
    matricula bigint PRIMARY KEY,
    alunoNome varchar(100) NOT NULL,
    cpf varchar(14),
    sexo integer NOT NULL,
    dtNascimento DATE NOT NULL,
    cidadeID_ integer NOT NULL,
    FOREIGN KEY(cidadeID_) REFERENCES cidades(cidadeID)
);

2.2 Localização e Ajuste da Connection String
Garanta que o servidor Firebird esteja rodando na porta padrão (ou na porta configurada).

localize seu arquivo de banco de dados (.FDB).

Localize e abra o arquivo appsettings.json no projeto EM.Web.

2.3 Após a criação das tabelas, você deve configurar o projeto para se conectar ao banco de dados:

Localize o Arquivo de Configuração: Abra o arquivo appsettings.json no projeto EM.Web.

Prepare o Caminho: Garanta que o servidor Firebird esteja rodando e encontre o caminho físico completo do seu arquivo .FDB.

Ajuste: Altere  para corresponder ao caminho e as credenciais:

"ConnectionStrings": {
  "FirebirdConnection": "User=SYSDBA;Password=masterkey; Database={Caminho físico do seu arquivo .FDB}; Port=3055(Porta padrão); Dialect=3; Charset=UTF8;"
}

Substitua `{Caminho físico do seu arquivo .FDB}` pelo caminho completo no seu sistema (ex: `C:\Dados\Meu_banco.FDB`). Altere o valor `Port=3055` apenas se o servidor Firebird estiver configurado em uma porta diferente.

--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

3. Inicialização e Execução
 Usando o Visual Studio (Recomendado)
 Abra a solução (PROJETO.EM.sln) no Visual Studio.

Defina o projeto EM.Web como o Projeto de Inicialização.

Pressione F5 ou clique no botão "Iniciar".

A aplicação será iniciada, e o navegador abrirá na rota padrão: /Aluno/ListarAlunos
